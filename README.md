# نظام متابعة المهام والاتصالات (CRM العقاري)

## Project Overview
نظام متابعة المهام والاتصالات هو تطبيقة ويب تستخدم لتسهيل وإدارة العلاقات مع العملاء، حيث يوفر واجهة مستخدم مريحة لمتابعة المهام والاتصالات الخاصة بالعقارات. تم تصميم التطبيق باستخدام تقنيات حديثة وسهلة الاستخدام، مما يساعد في تعزيز تجربة المستخدم.

## Installation
لتثبيت النظام على جهازك، يمكنك ببساطة تحميل ملف `index.html` وفتحه في متصفح الويب. لا يوجد برامج إضافية مطلوبة لتشغيل هذا النظام.

## Usage
بعد فتح `index.html` في متصفح الويب، ستظهر واجهة المستخدم. يمكنك التفاعل مع مختلف الأقسام مثل:
- **الرئيسية**: لرؤية ملخص عام عن الأنشطة والبيانات.
- **العملاء**: لإدارة معلومات العملاء.
- **المهام**: لمتابعة المهام المعلقة.
- **التقارير**: للحصول على تقارير محدثة عن الحالة الحالية.

## Features
- واجهة مستخدم متجاوبة وسهلة الاستخدام.
- قائمة تنقل بسيطة تسهل التنقل بين مختلف الأقسام.
- عرض داشبورد يحتوي على إحصائيات مهمة حول العملاء، الطلبات، الاجتماعات، والمهام المعلقة.
- قسم لمتابعة الأنشطة الأخيرة.

## Dependencies
هذا المشروع يعتمد على المكتبات التالية التي تم تضمينها في `index.html`:
- [Tailwind CSS](https://tailwindcss.com/) - إطار عمل CSS لتسهيل تصميم الواجهات.
- [Font Awesome](https://fontawesome.com/) - مكتبة أيقونات توفر مجموعة كبيرة من الأيقونات الجذابة.
- [Google Fonts](https://fonts.google.com/) - لتوفير خط Noto Kufi Arabic المستخدم في التطبيق.

## Project Structure
- `index.html`: الصفحة الرئيسية التي تحتوي على الكود HTML الخاص بالنظام.
- تستخدم المكتبات الخارجية مثل Tailwind CSS وFont Awesome، مما يجعل بنية المشروع بسيطة ومرنة.

### ملف index.html
- يحتوي على الهيكل الأساسي للصفحة، مع أقسام للنافيجاشن، الشريط الجانبي، ومحتوى الصفحات.
- يستخدم SVGs وCSS لتجميل العناصر وضمان توافر تجربة مستخدم ممتازة.
